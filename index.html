History

Back in the good old days, frontend landscape was different,
we used technologies like:

  • HTML
  • CSS
  • JS

JS was only for simple page intractions.

Like Submitting form data to server.
It work like that for a while.

But then There was a problem

Browsers were different!

maintained by different teams

and dealt differently with JS.

JS, modifies the DOM, and browsers didn't have backward compatibility.
Then Jquery saved the day!
It did the heavy lifting for the developer.
That power came with a problem

Developers started building huge applications.

Like Facebook

which jquery just couldn't handle!

Then Libraries like Backbone JS

Came into the picture

and SPA (Single Page Application) started to become a thing.
which means no more useless requests to the server to get HTML documents.
